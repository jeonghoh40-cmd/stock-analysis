# 📊 DART 기반 한국 유명 투자자 포트폴리오 확인 가이드

## 1️⃣ DART 전자공시란?

**DART**(Data Analysis, Retrieval and Transfer System) 는 금융감독원에서 운영하는 전자공시 시스템입니다.

- 🔗 공식 웹사이트: https://dart.fss.or.kr
- 📈 기업 공시, 재무제표, 대량보유보고서 등 확인 가능

---

## 2️⃣ 한국 유명 투자자 포트폴리오 확인 방법

### 📌 방법 1: DART 에서 직접 검색

#### 단계별 확인 절차:

1. **DART 접속**: https://dart.fss.or.kr

2. **검색창에 자산운용사명 입력**:
   - `체슬리투자자문`
   - `메리츠자산운용`
   - `라이프자산운용`
   - `VIP 자산운용`
   - `에셋플러스자산운용`

3. **공시 종류 필터**:
   - ✅ **대량보유보고서** (5% 이상 지분 보유 시 공시)
   - ✅ **투자신탁운용보고서** (분기별 펀드 보유 종목)
   - ✅ **증권신고서** (신규 펀드 출시 시)

---

### 📌 방법 2: 주요 공시 메뉴얼

#### 🔍 대량보유보고서 확인
```
1. DART 접속 → 검색창에 "라이프자산운용" 입력
2. "공시종류" 필터 → "대량보유보고서" 선택
3. 최근 공시 클릭 → 보유 종목 확인
```

#### 🔍 펀드운용보고서 확인
```
1. DART 접속 → 검색창에 "메리츠자산운용" 입력
2. "공시종류" 필터 → "투자신탁운용보고서" 선택
3. 분기별 보고서 클릭 → Top 10 보유 종목 확인
```

## 3️⃣ 투자자별 주요 확인 포인트

### 📌 박세익 (체슬리투자자문)
- **특징**: 반도체·2 차전지 사이클 투자
- **확인 공시**: 체슬리투자자문 운용 펀드 보고서
- **DART 검색**: `체슬리투자자문`
- **주력 분야**: 삼성전자, SK 하이닉스, LG 에너지솔루션

### 📌 존리 (메리츠자산운용)
- **특징**: 장기 가치투자, 금융주·플랫폼주
- **확인 공시**: 메리츠자산운용 투자신탁보고서
- **DART 검색**: `메리츠자산운용`
- **주력 분야**: 삼성전자, NAVER, 카카오, KB 금융

### 📌 이채원 (라이프자산운용)
- **특징**: 저 PBR, 지주사, 밸류업
- **확인 공시**: 대량보유보고서 (5% 이상)
- **DART 검색**: `라이프자산운용`
- **주력 분야**: 삼성전자, 기아, 현대차, CJ 제일제당

### 📌 김민국 (VIP 자산운용)
- **특징**: 행동주의, 밸류업, 지배구조 개선
- **확인 공시**: 대량보유보고서, 지분공시
- **DART 검색**: `VIP 자산운용`
- **주력 분야**: 현대건설, 한국조선해양, SK, GS

### 📌 강방천 (에셋플러스)
- **특징**: 장기 성장주, 소비재
- **확인 공시**: 펀드운용보고서
- **DART 검색**: `에셋플러스자산운용`
- **주력 분야**: 삼성전자, NAVER, 아모레퍼시픽

## 5️⃣ DART API 활용 (고급)

### API 키 발급
1. OpenDART 접속: https://opendart.fss.or.kr
2. 회원가입 → API 키 신청
3. 발급된 키를 `.env` 파일에 추가:
```
DART_API_KEY=your_api_key_here
```

### 주요 API 엔드포인트
- **회사개요**: `https://opendart.fss.or.kr/api/companyInfoList`
- **공시검색**: `https://opendart.fss.or.kr/api/list`
- **재무제표**: `https://opendart.fss.or.kr/api/fnlttMultiAcnt`
