## ② 기술적 분석 방법

### 📐 1. RSI (상대강도지수, 14 일)

```python
RSI = 100 - 100 / (1 + 평균상승폭 / 평균하락폭)

판단 기준:
  RSI < 20  → 과매도 (매수 신호, +40 점)
  RSI < 30  → 약과매도 (+25 점)
  RSI < 40  → 약세 (+10 점)
  RSI > 60  → 강세 (-10 점)
  RSI > 70  → 약과매수 (-25 점)
  RSI > 80  → 과매수 (매도 신호, -40 점)
```

---

### 📐 2. MACD (이동평균수렴확산지수)

```python
MACD 선 = 12 일 지수이동평균 - 26 일 지수이동평균
신호선 = MACD 의 9 일 지수이동평균
히스토그램 = MACD - 신호선

판단 기준:
  히스토그램 > 0  → 상승 모멘텀 (+20 점)
  히스토그램 < 0  → 하락 모멘텀 (-20 점)
```

---

### 📐 3. 이동평균 정배열

```python
MA5  = 5 일 이동평균
MA20 = 20 일 이동평균
MA60 = 60 일 이동평균

판단 기준:
  가격 > MA20 > MA60  → 강한 상승장 (+15 점)
  가격 > MA20         → 상승장 (+7 점)
  가격 < MA20 < MA60  → 강한 하락장 (-15 점)
  가격 < MA20         → 하락장 (-7 점)
  
MA5 > MA20  → 골든크로스 (+10 점)
MA5 < MA20  → 데드크로스 (-10 점)
```

---

### 📐 4. 모멘텀 (5 일)

```python
모멘텀 = (현재가 / 5 일전 가격 - 1) × 100

판단 기준:
  모멘텀 > +5%  → 강한 상승 (+10 점)
  모멘텀 > +2%  → 약상승 (+5 점)
  모멘텀 < -5%  → 강한 하락 (-10 점)
  모멘텀 < -2%  → 약하락 (-5 점)
```

---

### 📐 5. 볼린저 밴드 (BB)

```python
중심선 = 20 일 이동평균
상한선 = 중심선 + (2 × 표준편차)
하한선 = 중심선 - (2 × 표준편차)
BB 위치% = (현재가 - 하한선) / (상한선 - 하한선) × 100

판단 기준:
  BB 위치 < 10%  → 하한선 근접 (과매도)
  BB 위치 > 90%  → 상한선 근접 (과매수)
```

---

### 📐 6. 거래량 비율

```python
거래량비율 = 현재거래량 / 20 일 평균거래량

판단 기준:
  거래량비율 > 2.0  → 거래량 급증 (주목)
  거래량비율 < 0.5  → 거래량 위축
```
