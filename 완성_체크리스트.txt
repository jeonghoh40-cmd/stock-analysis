STEP 1. .env 파일에 이메일 정보 입력
─────────────────────────────────────
  .env 파일을 메모장으로 열어서 아래 4개 항목 수정:

    EMAIL_USER=본인Gmail@gmail.com
    EMAIL_PASS=앱비밀번호16자리      ← Gmail 앱 비밀번호 (아래 참고)
    EMAIL_FROM=본인Gmail@gmail.com
    (EMAIL_TO는 이미 geunho@stic.co.kr 로 설정됨)

  Gmail 앱 비밀번호 발급:
    1. myaccount.google.com 접속
    2. 보안 → 2단계 인증 ON
    3. 검색창에 "앱 비밀번호" 검색 → 새로 만들기
    4. 16자리 비밀번호 복사 → EMAIL_PASS 에 붙여넣기

STEP 2. (선택) 카카오톡 알림 설정
─────────────────────────────────────
  .env 파일의 KAKAO_TOKEN 에 액세스 토큰 입력
  → 방법은 설정_가이드.txt 참고

STEP 3. cron 자동 실행 등록
─────────────────────────────────────
  터미널(Terminal)을 열고:

    bash "폴더경로/setup_schedule.sh"

  → 이후 매일 오전 6:30에 자동 실행됩니다
